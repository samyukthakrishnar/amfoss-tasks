# Programming Language Run Commands

## Python
```bash
cd python
python3 python.py
```

## Java
```bash
cd java
javac java.java && java java
```

## C++
```bash
cd c++
g++ -o c++ c++.cpp && ./c++
```

## C
```bash
cd c
gcc -o c c.c && ./c
```

## Go
```bash
cd go
go run pygo.go
```

## Rust
```bash
cd rust
cargo run
# or if using the .rs file directly:
# rustc main.rs && ./main
```

## JavaScript
```bash
cd javascript
node javascript.js
```

## Ruby
```bash
cd ruby
ruby ruby.rb
```

## Elixir
```bash
cd elixir
elixir hollow_pyramid.ex
```

## Haskell
```bash
cd haskell
ghc haskel.hs && ./haskel
# or
runhaskell haskel.hs
```

## Note
Each language folder contains an `input.txt` file for Sub-Task 3 (reading number from file to create pyramid pattern).
